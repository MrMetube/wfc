# Wave Function Collapse auf Graphen

Diese App ist in der [Odin](https://github.com/odin-lang/Odin) Programmiersprache geschrieben.

## Build
```
odin run build
```
Es ist nicht garantiert das alle Teile des Build-Script(primär was Debugging angeht) auf anderen Geräten funktionieren. Die Grundfunktion  ohne weitere Parameter sollte aber funktionieren.


## Run
Die App lädt alle Beispiele(nur PNG) aus dem angegebenen Ordner.
```
.\build\wfc.exe <path/to/images/directory>
```
